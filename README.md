
## Focusnaut

https://github.com/user-attachments/assets/a054eb8f-9d76-4a50-807f-bf4c07c43201

O Focusnaut foi um Projeto 03 desenvolvido para o Hackathon do servidor no discord da Kiza Dev e pela Equipe 06. Consiste em uma plataforma com signin, signup, pomodoro e to do list simples. 

O foco foi em cada membro desenvolver features na área com mais afinidade, mas também colaborando no que soubesse em outras, fizemos toda a gestão das atividades usando Notion e o Board Project Manager no github com todos acessos disponíveis aos membros, para definir inssues com títulos e descrições mais claras, para caso alguém quisesse contribuir ou resolver as inssues. Além da documentação das ferramentas, features, bug fix, e contribuições.

Também priorizamos incentivar e ajudar uns aos outros principalmente quando solicitado pedidos de ajuda, e também aprender uns com os outro repassando seus aprendizados e experiências.

## Getting Started

#### Pre requisitos
1. Git e NodeJs Instalados
2. Instalar o PostgreSQL na sua máquina e configurar o usuário para para 'postgres' e definir um 'password' para ter acesso ao banco de dados.
3. no arquivo .env.exemple apague o .exemple e coloque sua senha do banco de dados no campo prédefinido.

#### 1. Fazer o clone do respositório do projeto para sua máquina local
``` bash
git clone https://github.com/tiobendev/focusnaut.git
```

#### 2. No seu terminal instale as dependências do frontend

```bash
## Instale as dependências no frontend
cd /focusnaut
npm install
ou
bun i

## Inicie o frontend
npm run dev

## Em outro terminal instale as dependências do backend
cd /focusnaut/backend
npm install
ou
bun i

## Inicie o backend
npm start
ou
bun start
```

Abra seu projeto usando [http://localhost:3000](http://localhost:3000) ou no link indicado no seu terminal para ver o projeto no seu browser.
